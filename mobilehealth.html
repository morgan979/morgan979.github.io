Mobile Health

Identifying depression from phone sensor data

Skills

Machine Learning
Weka
Python

Problem and Opportunity

People's depressive symptoms may be less prominent at work
  • Researchers were attempting to establish effective anti-depression intervention strategies for individuals while at work
  • Analyzing phone sensor data has the potential to automatically detect if an individual is at work and respond accordingly

Our Work

Applied machine learning techniques to phone sensor data from the Feinberg School of Medecine
  • Familiarized ourselves with the phone sensor data from 207 participants
  • Cleaned and normalized the data using a python script
  • Created 129 features to identify the workday label using python scripts
  • Used Weka to compare the accuracies of different classifiers

Results and Future Work

Correlation between phone sensor data and workday
  • Analyzed the accuracy, ROC area, F-measure, precision, and recall to find the set of data that best predicted workday versus non-workday
  • Random forest classifier with 150 trees consistently reported the highest accuracy
  • In the future, identify a label in addition to workday that may be linked to depression, such as the amount of time spent at home
  • Creating another study to gather less biased phone sensor data may improve the ability to determine depression
